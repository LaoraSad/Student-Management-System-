# Student Management System with Data Persistence

---

## description
A simple system that allows you to manage basic student information. The system can register, view, update, and delete students.

---

## Features
*Register students in a database
*View registered students
*Update student information
*Delete a student from the database


## 🖥️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/LaoraSad/Student-Management-System-.git
```

2. Navigate to the project folder:

```bash
cd init
```

3. Run the game:

```bash
python 3 inicio.py
```
